# 10 나만의 QR 코드 생성기 쉽게 만들기

링크와 중앙 이미지를 입력하면, 로고가 들어간 QR 코드를 한꺼번에 만들어 지정 폴더에 저장하는 Python 프로그램입니다.

---

## 개요

이 프로젝트는 **Gradio**로 친절한 웹 UI를 제공하고, **qrcode**와 **Pillow**로 QR 코드를 생성합니다. 사용자가 업로드한 이미지는 QR 코드 중앙에 흰색 배경 위에 배치되며, 링크는 최대 10개까지 추가할 수 있습니다.

**주요 기능**

- 링크 입력 (복사·붙여넣기 지원)
- 중앙 로고 이미지 업로드
- 로고 뒤 QR 패턴이 보이지 않도록 흰색 배경 처리
- `+ 링크 추가` 버튼으로 최대 10개 링크 입력
- **QR 코드 생성** 버튼으로 미리보기 확인
- **저장** 버튼으로 Finder에서 폴더를 선택해 한꺼번에 PNG 저장

**사용 기술**

| 구분 | 라이브러리 / 도구 |
|------|-------------------|
| UI | Gradio |
| QR 생성 | qrcode |
| 이미지 처리 | Pillow |

## 설치

프로젝트 폴더로 이동한 뒤 필요한 패키지를 설치합니다.

```bash
cd "10 나만의 QR 코드 생성기 쉽게 만들기"
pip3 install -r requirements.txt
```

`requirements.txt`에 포함된 패키지:

```
gradio>=4.44.0
qrcode[pil]>=7.4.2
Pillow>=10.0.0
```

## 사용법

### 1. 앱 실행

```bash
python3 qr_generator.py
```

터미널에 표시되는 주소(예: `http://127.0.0.1:7860`)를 브라우저에서 엽니다.

### 2. QR 코드 만들기

1. **중앙에 넣을 이미지** 업로드
2. **링크 1**에 URL 입력 (복사·붙여넣기 가능)
3. 링크가 더 필요하면 **+ 링크 추가** 클릭 (최대 10개)
4. **QR 코드 생성** 버튼 클릭 → 미리보기 확인
5. **저장** 버튼 클릭 → Finder 폴더 선택 창에서 저장 위치 선택

생성된 파일은 `qr_01_example.com.png` 형식으로 저장됩니다.

### 3. 실행 예시

```bash
python3 qr_generator.py
```

브라우저에서 앱을 연 뒤 링크 `https://www.google.com`과 중앙에 넣을 로고 이미지를 업로드하고 **QR 코드 생성**으로 미리보기를 확인한 다음, **저장**을 눌러 폴더에 PNG 파일을 저장합니다.

## 동작 설명

| 항목 | 설명 |
|------|------|
| 중앙 이미지 | QR 코드 너비의 약 1/5 크기로 리사이즈 후 정중앙 배치 |
| PNG 로고 처리 | PNG는 투명 여백 제거 후 비율 유지, 필요한 경우에만 축소(확대 없음) |
| 흰색 배경 | 로고 주변에 흰색 사각형을 깔아 뒤 QR 패턴이 비치지 않게 처리 |
| 오류 복원 | 중앙 로고 때문에 가려지는 영역을 위해 `ERROR_CORRECT_H` 사용 |
| 링크 형식 | `http://` 또는 `https://`가 없으면 자동으로 `https://` 추가 |
| 파일명 | `qr_01_도메인.png` 형식으로 순번 저장 |
| 저장 방식 | **QR 코드 생성** 후 **저장** 버튼 클릭 시 Finder 폴더 선택 창에서 위치 지정 |

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 패키지 설치 오류 | `pip3 install -r requirements.txt` 다시 실행 |
| 폴더 저장 실패 | 선택한 폴더의 쓰기 권한 확인 |
| QR 스캔이 잘 안 됨 | 로고가 너무 크면 인식이 어려울 수 있음. 더 단순한 이미지 사용 권장 |
| 저장 버튼이 비활성 | 먼저 **QR 코드 생성** 버튼을 눌러 주세요 |
| 링크 추가 버튼 비활성 | 이미 10개까지 추가된 상태 |
